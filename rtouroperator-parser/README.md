# RTourOperator Parser

Парсер туров с сайта rtoperator.ru

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python rtour_parser.py
```

## Что парсится

1. **Список туров** с главной страницы:
   - Название тура
   - ID тура
   - Ссылка на тур
   - Маршрут (города)
   - Длительность (дни)
   - Базовая цена
   - Валюта
   - Изображение

2. **Даты и ведомости** для каждого тура:
   - Конкретные даты заездов
   - Цена на дату
   - Наличие мест

## Структура данных

Результат сохраняется в `rtoperator.json`:

```json
{
  "source": "rtoperator",
  "parsed_at": "2025-11-18T...",
  "total_count": 150,
  "programs": [
    {
      "id": 12345,
      "name": "Название тура",
      "url": "https://www.rtoperator.ru/...",
      "route": ["Москва", "Казань"],
      "days": 3,
      "price": 15000,
      "currency": "руб",
      "image": "https://...",
      "dates": [
        {
          "date": "01.01.2026",
          "price": 15500,
          "available": true
        }
      ]
    }
  ]
}
```

## Особенности

- **Пагинация**: Автоматически определяет количество страниц из скрытого `<div style="display: none">`
- **Параллельная загрузка**: Страницы парсятся параллельно для ускорения
- **Async/await**: Использует асинхронные запросы через aiohttp
- **Защита от блокировки**: Полноценные HTTP заголовки браузера
