# Быстрый старт

## Вариант 1: Запуск через веб-интерфейс (Рекомендуется)

### Windows:
Просто запустите файл `start.bat`

### Или вручную:
```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Запустите сервер
python app.py

# 3. Откройте в браузере
# http://127.0.0.1:5000
```

## Вариант 2: Использование через командную строку

```python
# Отредактируйте список запросов в ad_parser.py
# и запустите:
python ad_parser.py
```

## Пример использования

### Через веб-интерфейс:

1. Запустите `start.bat` или `python app.py`
2. Откройте http://127.0.0.1:5000
3. Введите запросы:
   ```
   купить iPhone
   доставка пиццы москва
   ремонт квартир
   ```
4. Нажмите "Запустить парсинг"
5. Дождитесь окончания
6. Скачайте результаты

### Результат будет содержать:
- Заголовки объявлений
- Описания
- URL сайтов
- Типы рекламы (контекстная/поисковая)
- Время сбора данных

## Требования

- Python 3.7+
- Google Chrome
- Интернет-соединение

## Структура проекта

```
Яндекс бизнес/
├── app.py                  # Веб-приложение Flask
├── ad_parser.py           # CLI версия парсера
├── templates/
│   └── index.html         # Веб-интерфейс
├── requirements.txt       # Зависимости
├── start.bat              # Скрипт запуска (Windows)
└── README.md             # Документация
```

## Устранение проблем

**Не запускается сервер:**
```bash
pip install --upgrade flask selenium webdriver-manager
```

**ChromeDriver не найден:**
- Установится автоматически при первом запуске
- Требуется наличие Google Chrome

**Captcha при парсинге:**
- Увеличьте задержки между запросами
- Используйте меньше запросов за раз
- Рассмотрите использование прокси

## Поддержка

При возникновении проблем проверьте:
1. Установлен ли Python 3.7+
2. Установлен ли Google Chrome
3. Установлены ли зависимости из requirements.txt
4. Есть ли доступ к интернету
