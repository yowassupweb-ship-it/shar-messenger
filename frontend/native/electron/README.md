# Electron client

Electron-клиент открывает удалённую веб-версию приложения в отдельном нативном окне.

## Настройка

1. Скопируйте `.env.example` в `.env.local`
2. Укажите `REMOTE_WEB_URL`
3. Установите зависимости: `npm install`
4. Запуск: `npm run dev`
5. Сборка Windows: `npm run build`

## Переменные окружения

- `REMOTE_WEB_URL` — URL опубликованного фронтенда, который уже работает с удалённым бекендом

## Почему так

Текущий фронтенд построен на Next.js App Router и активно использует свои маршруты и относительные `/api/*` вызовы. Поэтому для быстрого и безопасного переноса в нативную оболочку Electron сейчас используется webview-подход без переписывания экранов.
