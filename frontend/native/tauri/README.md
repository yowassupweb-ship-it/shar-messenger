# Tauri client

Tauri-клиент запускает удалённую веб-версию приложения в нативной оболочке и подходит для Windows и Android.

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Укажите `REMOTE_WEB_URL`
3. Установите зависимости: `npm install`
4. Для Windows: `npm run dev`
5. Для Android: сначала `npm run android:init`, потом `npm run android:dev`

## Переменные окружения

- `REMOTE_WEB_URL` — URL опубликованного фронтенда, который уже подключён к удалённому бекенду

## Примечания

- Android-сборка требует установленный Android SDK и Rust mobile target.
- Идентификатор приложения уже задан в конфиге Tauri и готов для Windows/Android.
- Сейчас используется оболочка над удалённым веб-приложением, чтобы сохранить все текущие Next.js маршруты без переписывания экранов.
