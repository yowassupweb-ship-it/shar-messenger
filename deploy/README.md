# Деплой скрипты Shar OS

## `.shar-deploy.ps1`

Полный автоматический деплой Electron приложения + обновление сервера одной командой.

### Использование

```powershell
# Из папки deploy
cd deploy
.\.shar-deploy.ps1              # Patch версия (0.2.6 -> 0.2.7)
.\.shar-deploy.ps1 -Minor       # Minor версия (0.2.6 -> 0.3.0)
.\.shar-deploy.ps1 -Major       # Major версия (0.2.6 -> 1.0.0)
.\.shar-deploy.ps1 -Version 2.0.0  # Конкретная версия

# Или из корня проекта
.\deploy\.shar-deploy.ps1
```

### Что делает скрипт:

1. **Обновляет версию** в `frontend/native/electron/package.json`
2. **Собирает Electron приложение** (Windows NSIS installer)
3. **Загружает файлы на сервер** (установщик, обновления, метаданные)
4. **Обновляет сервер** командой `shar pull` (таймаут: 5 минут)
5. **Коммитит изменения** в Git и пушит на GitHub

### Требования

- PowerShell 5.1+
- Переменная `SSH_PASSWORD` в файле `.env` в корне проекта
- Установленный npm (для сборки)
- Доступ к серверу по SSH

### Особенности

- **Без предупреждений SSH**: все SSH warnings подавлены
- **Увеличенный таймаут**: команда `shar pull` может работать до 5 минут
- **Чистый вывод**: Git операции не показывают лишних сообщений
- **Автофикс latest.yml**: автоматическое копирование правильной версии из nsis-web

### Примечания

- Скрипт автоматически исправляет проблему с `latest.yml` (копирует из `nsis-web`)
- Все файлы загружаются на сервер в `/var/www/shar/updates`
- Команда `shar pull` обновляет веб-приложение на сервере
- Время выполнения: ~2-3 минуты (зависит от скорости сети)
